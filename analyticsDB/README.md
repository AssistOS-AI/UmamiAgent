# Analytics DB

`analyticsDB` is the Ploinky-managed PostgreSQL service for the Umami analytics stack.

It runs `postgres:16-alpine`, joins the `analytics` network as `analytics-db`, and persists data in `.ploinky/data/analyticsDB/postgres`.

The shared `UMAMI_POSTGRES_PASSWORD` secret is generated by Ploinky and reused by `analyticsInfra`.
